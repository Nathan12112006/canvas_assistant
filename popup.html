Settings and proxy connection
Private setup
1
Find your Canvas domain. It is the part before /courses in your URL, for example scu.instructure.com.
2
Go to Canvas -> Account -> Settings -> Approved Integrations -> New Access Token, then paste it below.
3
Run your private AI proxy and paste its URL below. The proxy holds the OpenAI API key so users never see it in the extension.
Canvas Domain
